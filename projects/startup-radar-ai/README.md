# Startup Radar AI

Находит 20 AI-стартапов, анализирующих рынок стартапов, и формирует сравнительную таблицу с бизнес-моделью, ключевыми метриками и moat.

## Что делает

1. Генерирует 6 поисковых запросов через Claude
2. Ищет свежие упоминания через Exa Search (фильтр по дате)
3. Проверяет релевантность каждого кандидата
4. Извлекает: бизнес-модель, метрики, moat
5. Сохраняет markdown-отчёт с таблицей

## Требования

- Python 3.9+
- Exa API key ([exa.ai](https://exa.ai/)) — free tier: 1 000 запросов/месяц
- Claude подписка на [claude.ai](https://claude.ai) — для финального анализа

## Установка

```bash
pip install exa-py python-dotenv
cp .env.example .env
# Вставить EXA_API_KEY в .env
```

## Запуск

```bash
python src/main.py
```

## Как это работает

1. Скрипт ищет кандидатов через Exa Search
2. Фильтрует по ключевым словам (AI + startup)
3. Сохраняет данные в `output/candidates_YYYY-MM-DD.json`
4. Генерирует готовый промпт в `output/prompt_YYYY-MM-DD.txt`
5. Ты вставляешь промпт в claude.ai → получаешь таблицу

## Параметры (`src/config.py`)

| Параметр | По умолчанию | Описание |
|---|---|---|
| `TOPIC` | AI startup analytics | Тема поиска |
| `DAYS_BACK` | 60 | Период поиска в днях |
| `TARGET_COUNT` | 20 | Желаемое количество стартапов |
| `STAGE_FILTER` | seed, pre-seed, early | Фильтр по стадии | первый коммит с телефона
