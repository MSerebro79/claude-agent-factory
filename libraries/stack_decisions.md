# Stack Decisions

Накопленные решения по выбору стека.
Пополняется после каждых 3–5 проектов.

---

## Когда что использовать

| Задача | Решение | Почему |
|---|---|---|
| Хранение данных | Supabase | Postgres + API из коробки, бесплатный tier |
| Векторный поиск | Qdrant | Self-hosted на VPS, без внешних зависимостей |
| Автоматизация | n8n | Self-hosted, визуальный редактор, легко дебажить |
| Уведомления | Telegram Bot | Быстро, бесплатно, всегда под рукой |
| Веб-поиск | Exa | Семантический поиск, API простой |
| LLM | Claude Sonnet | Баланс качества и стоимости |
| Язык кода | Python | Экосистема AI-инструментов |

---

## Когда НЕ использовать агента

| Ситуация | Альтернатива |
|---|---|
| Один линейный процесс без ветвлений | Python скрипт |
| Регулярная задача по расписанию без LLM | n8n workflow |
| Форма + сохранение в базу | Supabase + простой скрипт |
| Вопрос-ответ по документу | Claude напрямую |
