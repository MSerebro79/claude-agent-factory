# Cost Library

Справочник по стоимости компонентов стека.
Используется Designer при заполнении Cost Estimation в blueprint.

---

## LLM

| Модель | Стоимость | Примечание |
|---|---|---|
| Claude Sonnet (claude-sonnet-4-20250514) | $3 / 1M input tokens, $15 / 1M output | Основной выбор |
| Claude Haiku | $0.25 / 1M input, $1.25 / 1M output | Для простых задач |

Типовые расчёты:
- 100 запросов/день × 2K токенов = ~$18/месяц на Sonnet
- 10 запросов/день × 2K токенов = ~$1.8/месяц на Sonnet
