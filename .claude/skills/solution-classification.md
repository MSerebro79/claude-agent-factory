# Solution Classification

Используй эту процедуру для определения правильного типа решения.

## Таблица типов решений

| Тип | Когда использовать | Примеры |
|---|---|---|
| **Prompt** | Разовая аналитика, нет автоматизации | Анализ документа, summary |
| **Script** | Повторяемая логика, запустил → получил результат | Парсинг CSV, пересчёт |
| **Workflow** | Триггеры, интеграции, расписание | RSS → Telegram, cron |
| **Agent** | Анализ неструктурированных данных, ветвления | Research, оценка |
| **Multi-Agent** | Несколько независимых ролей, параллельная работа | Сложный pipeline |

## Пошаговые вопросы

| Вопрос | Да | Нет |
|---|---|---|
| Задача разовая, без автоматизации? | **Prompt** | далее |
| Workflow повторяемый, структурированные данные, нет LLM? | **Script** | далее |
| Нужны триггеры, интеграции, расписание? | **Workflow** | далее |
| Нужен анализ или синтез неструктурированных данных? | **Agent** | Script/Workflow |
| Следующий шаг неизвестен заранее? | **Agent** | Script/Workflow |
| Можно решить одним промптом? | **Prompt** | далее |
| Несколько независимых ролей? | **Multi-Agent** | Single Agent |

## Действия по результату

**Prompt:** решается напрямую в Claude, строить ничего не нужно.
**Script / Workflow:** используй `templates/script_blueprint_template.md`
**Agent / Multi-Agent:** переходи к Agent Qualification Test ниже.

Обязательно заполни секцию `6b. Alternative Solutions` в blueprint — почему нельзя
решить проще. Пример для Research Agent:
- Prompt: нет — нужна память и повторный поиск
- Script: нет — требуется анализ неструктурированных данных
- Workflow: нет — нужна адаптация логики
- Agent: ✓

## Agent Qualification Test

*(только для Agent и Multi-Agent)*

| # | Вопрос | Да | Нет |
|---|---|---|---|
| 1 | Workflow непредсказуемый, есть ветвления? | ок | скрипт/n8n |
| 2 | Нужен анализ неструктурированных данных или синтез? | ок | скрипт/n8n |
| 3 | Следующий шаг зависит от результатов анализа? | ок | скрипт/n8n |
| 4 | Нельзя решить одним промптом напрямую? | ок | Prompt |
| 5 | Экономия времени > 5 часов в неделю? | ок | красный флаг |
| 6 | Каждый предполагаемый агент имеет уникальную роль? | ок | объединить |

Если ≥4 ответов "нет" → пересмотри Solution Type.
Если решение = один вызов LLM без цикла → это Prompt, не Agent.

Запиши результат в blueprint секцию `7a. Agent Justification`.
